Tree Shakeable App
------------------------------

## How to start

```
# install all dependencies.
$ npm ci

# Start developing and serve your app at port 9000:
$ npm run dev
```

Output should be available in here: http://localhost:9000/main.bundle.js

## How to use

```
# Start developing and serve your app at port 9000:
$ npm run dev

# Build your app: 
$ npm run build

# run unit tests:
$ npm run test
```
